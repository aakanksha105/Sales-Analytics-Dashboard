# Sales Analytics Dashboard

## 📊 Overview
This project analyzes sales data to uncover trends, customer behavior, and overall business performance using an interactive Power BI dashboard.

## 🧠 Problem Statement
To analyze sales performance and identify key drivers of revenue, customer segmentation, and regional trends to support data-driven business decisions.

## 📁 Dataset
The dataset includes:
- Sales transactions
- Customer information
- Product categories
- Region and time-based attributes

## ⚙️ Approach
- Cleaned and transformed raw data using Power Query
- Built a relational data model connecting sales, customer, and product tables
- Created calculated columns and DAX measures for business KPIs
- Performed exploratory data analysis to identify trends and patterns

## 📊 Dashboard Features
- Revenue trends over time (time-series analysis)
- Sales performance by region and category
- Customer segmentation analysis
- KPI cards for revenue, sales, and performance metrics
- Interactive filters (region, time, customer type)

## 🔄 Dashboard Interactivity
- Enabled dynamic filtering using slicers for region, year, and customer segments
- Implemented drill-down capabilities for deeper analysis
- Designed user-friendly layout for intuitive data exploration

## 🔍 Key Insights
- Identified top-performing regions and product categories driving revenue
- Highlighted underperforming segments requiring business attention
- Observed patterns in customer purchasing behavior across regions and time
- Detected seasonal trends impacting sales performance

## 🚀 Impact
- Supports data-driven decision-making for business strategy
- Improves visibility into key performance metrics
- Helps identify growth opportunities and operational inefficiencies
- Enables stakeholders to monitor performance interactively

## 🛠 Tools Used
- Power BI
- SQL
- Power Query
- DAX
