# Sales Analytics Dashboard

## 📊 Overview
This project focuses on analyzing sales data to uncover trends, customer behavior, and business performance using Power BI dashboards.

## 🧠 Problem Statement
To analyze sales performance and identify key factors affecting revenue, customer segmentation, and regional performance.

## 📁 Dataset
The dataset includes:
- Sales transactions
- Customer details
- Product categories
- Region and time-based data

## ⚙️ Approach
- Performed data cleaning and transformation using Power Query
- Designed data models with relationships between tables
- Created calculated columns and measures
- Built KPIs such as:
  - Revenue
  - Sales performance
  - Customer segmentation

## 📊 Dashboard Features
- Revenue trends over time
- Sales by region
- Customer segmentation
- KPI cards for business performance

## 🔍 Key Insights
- Identified top-performing regions and products
- Highlighted underperforming segments
- Observed trends in customer purchasing behavior

## 🚀 Impact
- Enables better business decision-making
- Improves visibility into sales performance
- Supports strategic planning and forecasting

## 🛠 Tools Used
- Power BI
- SQL
- Power Query
